# Easy Electrical Website

This is a simple static website for Easy Electrical.

## Files
- `index.html` — main site entry point
- `mepfs.html` — alternate page copy using the same content
- `styles.css` — styling for the site
- `script.js` — mobile menu toggle behavior
- `logo.svg` — brand mark for the header

## Deploying
1. Open `index.html` in a browser to preview locally.
2. Upload the files to any static hosting provider.
   - GitHub Pages
   - Netlify
   - Vercel
   - any shared hosting / file server

## Notes
- The site is responsive and includes a mobile menu.
- Use `index.html` as the published home page.
